# React Users Data

A simple React app that fetches Users from a public API and displays them in a responsive UI.

## Features
- Fetch data from API
- Loading & error handling
- Responsive design
- Clean UI with reusable components

## Tech Stack
- React
- Vite

## API Used
https://jsonplaceholder.typicode.com/users

## Run Locally
npm install  
npm run dev
